・このファイルの中身について

　こちらは、2Dアクションゲーム、アップルゲッターという作品になっております。大学の講義にて作成した作品となっております。
最初のタイトル画面でスペースキーを押すとゲームがスタートし、60秒間振ってくる爆弾に気をつけながらリンゴを獲得する内容になっております。Zキーでジャンプし、十字キーで移動します。
ゲーム内にはスコアが用意されており、30点以上取るとクリアとなり、それ以下だとゲームオーバーと表記されます。また、クリアするかゲームオーバーになった後も繰り返しプレイできます。さらに、時間が経過すると振って来るリンゴと爆弾の落下速度が速くなる要素もあります。

工夫点として、一部の画像を自分で作成し、講義でやった部分だけでなく、自力で機能を追加するように心がけたことです。（講義では時間制限が来たらゲームオーバー･･･で完成という事になっていましたが、私はさらに、スコアが一定量超えたらクリアと表記されるように機能を追加いたしました。）

反省点としては、本来はステージを複数作ろうと考えていましたが、実装できなかった事。他にもゲームを繰り返しプレイした際、落下速度が速いままで始まってしまうなど、修正しきれなかった部分もあります。
